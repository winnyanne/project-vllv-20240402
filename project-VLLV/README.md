## 專案技術

- Node.js v16.15.0
- Vite v4.0.4
- Axios v0.24.0
- Bootstrap v5.1.3


## 功能簡介

此專案有使用 Github Actions，所以發起 PR 時會自動執行以下動作：

- 建立 Node.js 環境
- 安裝相依套件
- 編譯程式碼
- 執行 ESLint 掃描
- 執行測試
