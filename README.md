## 功能簡介
瀏覽 SPA 單頁式網頁應用 < 失戀疫苗 > 課程活動 
註冊並登入 進入 會員專區
提供：介紹課程內容、回顧課程演變、
商品選購->點擊商品的購物車圖示(紅色加入、黑色尚未加入)
點擊購物車->購物清單->設定商品數量
輸入購買人資訊->提交
會員專區->訂單資料->查看訂單狀態
